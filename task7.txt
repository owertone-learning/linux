########### 4. * Настроить проброс портов локально с порта 80 на порт 8080.

marina@marina-HP-Linux:~/students$ sudo iptables -t nat -A PREROUTING -p tcp --dport 80 -j REDIRECT --to-ports 8080

marina@marina-HP-Linux:~/students$ sudo iptables -t nat -L
Chain PREROUTING (policy ACCEPT)
target     prot opt source               destination         
REDIRECT   tcp  --  anywhere             anywhere             tcp dpt:http redir ports 8080
